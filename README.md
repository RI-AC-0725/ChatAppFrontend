# name

ChatAppFrontend

## Recommended IDE Setup

- VSCode 1.97.1
- node 22.13.1
- npm 10.9.2

## Overview

複数人で決められたテーマについてチャットできるWEBアプリ

## Usage

興味のあるテーマでチャットができる
好きなテーマを決めてチャットルームを作成できる
自分だけしか投稿出来ない個室を作ることができる
